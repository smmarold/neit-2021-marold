Software Layout, 2021

The Software Layout project was my first foray into the world of application and software design. The primary objective was to work with the CSS and JavaScript to make funtional buttons, tabs, and resizing columns as needed. While simple, the project was an important stepping stone in my understanding of software, layouts, CSS and JavaScript.

    Some of the concepts covered here were:
  • Selecting elements using JavaScript
  • Assigning Elements classes through the JS to change the look and layout of the software.
  • Using CSS to show/hide elements, like a popout sidebar.
  • Practice creating a 1-1, functional copy of a given design.
  • Formatting and positioning objects in CSS using Flexboxes.
